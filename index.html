TAREAS ADMINISTRATIVAS Y SEGURIDAD

CAMBIO DE CONTRASEÑA

Cualquier usuario puede cambiar su contraseña usando el comando "passwd", de la siguiente manera: Contraseña Tenga en cuenta que el comando no requiere argumentos. El administrador puede cambiar la contraseña de cualquier usuario. Por ejemplo, suponga que el administrador tiene que cambiar la contraseña de un usuario cuyo login es lisa. Esto se puede lograr escribiendo lo siguiente dominio: contraseña lisa Una cosa interesante es que el sistema operativo obliga a los usuarios habituales a crear contraseñas largas, con un mínimo de 5 o 6 caracteres; mientras que el administrador puede crear contraseñas de cualquier longitud. Para ejemplo, si un usuario normal intenta crear la contraseña "123", el sistema no lo permitirá; pero el administrador podría hacerlo.

CREAR Y ELIMINAR USUARIOS

Solo los administradores pueden crear y eliminar cuentas. Para crear una cuenta, use el comando "adduser". Por ejemplo, para crear una cuenta cuyo nombre de usuario (iniciar sesión) es lisa, usa el comando: agregar usuario lisa Solo los administradores pueden crear y eliminar cuentas. Para crear una cuenta, use el comando "adduser". Por ejemplo, para crear una cuenta cuyo nombre de usuario (iniciar sesión) es lisa, usa el comando: agregar usuario lisa Este comando crea automáticamente el directorio de inicio y el grupo del usuario, ambos llamados lisa. Además, durante la ejecución del comando, el sistema operativo solicitará la introducción del contraseña que se utilizará para ingresar a la cuenta, así como otra información menos relevante como: full nombre del usuario, número de teléfono, etc. Por otro lado, para eliminar una cuenta se utiliza el comando “deluser”. Por ejemplo, para eliminar un cuenta cuyo nombre de usuario (login) es lisa, use el comando: deluser lisa Con este comando se elimina el grupo principal de la cuenta, es decir, el grupo lisa. Pero, el hogar permanece el directorio ubicado en el directorio de inicio. Para eliminar el directorio de inicio debemos agregar el “-- opción remove-home”, de la siguiente manera: deluser --remove-home lisa En cualquier caso, el administrador siempre puede eliminar el directorio de inicio de la cuenta mediante el comando: rm-r lisa

CREAR Y ELIMINAR GRUPOS

Solo los administradores pueden crear y eliminar grupos. Para crear un grupo, utilice el comando "groupadd". Por ejemplo, para crear un grupo llamado estudiantes vamos a use el siguiente comando: agrupar estudiantes Por otro lado, para eliminar grupos se utiliza el comando “groupdel”. Por ejemplo, para eliminar al estudiante grupo usaremos el siguiente comando: grupo de estudiantes

AGREGAR Y ELIMINAR USUARIOS A GRUPOS

Solo los administradores pueden agregar y eliminar usuarios de los grupos. Para hacer que un usuario pertenezca a varios grupos, además del propio, se utiliza el comando "usermod" con las opciones a y G. Por ejemplo, para que la usuaria lisa pertenezca a los grupos de profesores y tutores, la Se usarán los siguientes comandos usermod -aG profesores lisa usermod -aG tutores lisa Estos dos comandos se pueden reducir a uno separando los nombres de los grupos con comas: usermod -aG profesores, tutores lisa Tenga en cuenta que no hay espacios en blanco entre los nombres de los grupos y la coma de separación. Por otro lado, para eliminar un usuario de un grupo el comando "deluser". Por ejemplo, para quitar el usuario cuyo inicio de sesión es lisa del grupo de tutores, use el siguiente comando: deluser tutores lisa

CAMBIAR EL GRUPO DE UN ARCHIVO O DIRECTORIO

By default a file or directory belongs to the group of the user who creates it. To change to the group of the file, use the “chgrp” command. For example, to make the test.txt file belong to the group teachers, use the command: chgrp teachers test.txt To change the group of a directory and all its content, the R option is used. For example: chgrp -R teachers test makes the directory test and all its content belong to the group teachers.
